Общее затраченное время - 8ч

Для запуска проекта нужно иметь любой локальный сервер (OpenServer, xampp и д.р) или установить стек LAMP.

После установки и настройки сервера, скачайте проект ссылка. Скачанный проект перенесите в дирректорию созданного домена.

Создайте MySql базу данных. В файле настроек .env установите следующим константам ваши значение
DB_HOST
DB_DATABASE
DB_USERNAME
DB_PASSWORD

Выполните следующие команды:
composer update
php artisan migrate

В проекте три метода. 
user_auth - метод для авторизации пользователей. Основной метод. Передача данных GET. Параметр: data.

Следующие методы нужны исключительно для облеглегчения подготовки данных.
sig - метод пригимает в себя следующие параметры id, first_name, last_name, city, country, access_token. Вернет подготовленный sig.
spr - этот метод нужен для имитация подготовки отправляемых данных со стороны фронта. Передача данных POST. 
Параметры: access_token, id, first_name, last_name, city, country, sig.

Обращение к методам: http://your_domain/api/user_auth

Первым нужно выполнять метод sig. Полученный результат установить параметру sig в методе spr. Метод spr вернет подготовленный хэш, его нужно установить параметру data в методе user_auth.
